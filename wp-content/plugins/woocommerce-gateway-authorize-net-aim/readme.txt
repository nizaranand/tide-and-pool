== Installation ==

 * Unzip the files and upload the folder into your plugins folder (wp-content/plugins/) overwriting old versions if they exist
 * Activate the plugin in your WordPress admin area.
 * Open the settings page for WooCommerce and click the gateways tab
 * Configure your Authorize.net settings.  See below how to.
 
== Important Note ==

You *must* enable SSL from the settings panel to use this plugin in live mode - this is for your customers safety and security.


== Where to find your Autorize.net Credentials ==

To setup your Autorize.net payment gateway you will need to enter your API Login ID and Transaction Key.
1.  Login to your Authorize.net account
2.  Click on the Account tab on the top navigation
3.  Click on the Settings tab on the left navigation
4.  Under the Security Settings area, click on "API Login ID and Transaction Key"
5.  Your API Login ID will be displayed, copy this value to your WooCommerce settings panel
6.  To get the Transaction Key answer the Secret Question (you can check the box next to "Disable Old Transaction Key(s)")
7.  Press Submit and your current Transaction Key will be displayed, copy this value to your WooCommerce settings panel
8.  Save the settings.
